Get Random Images

You can get Random Images from different sources, at the moment, we have implemented just Flickr and Google.

GET /images/:source

:source => The source of the service to load the images from, at the moment we support "Google" or "Flickr"

Add Favourites

Add an image to favourite, you can add one an image that you have got from our service or you can just pass a URL of an image

POST /favourite

POST parameter
url => url of the image

Valid Response Example
{"response":true}
Error Response Example
{"response":false, "message":"The url provided is not valid"}

Remove Favourite

You don't like an image anymore? You can delete it

DELETE /favourite/:id

:id => of the image that you want to get rid of

Valid Response Example
{"response":true}
Error Response Example
{"response":false, "message":"Something went wrong"}

Edit Favourite

You don't like an image anymore? You can delete it

PUT /favourite/:id

:id => of the image that you want to get rid of

POST parameter
description => description of the image

Valid Response Example
{"response":true}
Error Response Example
{"response":false, "message":"Something went wrong"}
