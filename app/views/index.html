Welcome to SimpleGallery

A small webpage that loads images from Google or Flickr (at the moment) and they can be added to the Favourites.

Why have I done this webservice?

Someone requested it to me and I just did it.

How I did it?

Since I had a lot of time, I have decided to make a good structure in the project because if you don't start a project well then it will always continue following the bad way. The written code is PHP5.3 and it uses namespaces.
I have decided to use a Dependency Injection library called Pimple which is written by Fabien Potencier, the creator of Symfony and I really like this library because it's simple, it uses closures and it makes the stuff done.
Then I decided to use KLogger, becuase I think that we (developers), should log everything that happens in our app.
For the Database I used just a simple PDO, which is good but if it was a bigger project I would have choosen Doctrine, a very powerfull ORM.
I used a router, that I found on internet long time ago but now I could not find who created it but I have modified it a little bit.
Since I am not good at design I decided to use a template, because I think the eye needs his part on everything. The template was made by Jonathan Holroyd.

Around all this stuff I've built a little MVC, so I could be more clean, I have separated the code from the UI thanks to ANETple (a Template Engine written by me long time ago). Using the Router was easy for me to implement some APIs as well. Infact we can see the APIs in the APIs Section. There you can find some documentation and a PHP client with everything already implemented.
